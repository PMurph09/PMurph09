## What I've Been Cooking Up (More Coming Soon!)

### [Push-T: Behavioral Cloning Demo](https://github.com/PMurph09/t-push)
Interactive web demo comparing three behavioral cloning agents — MLP, CVAE, and Diffusion (DDPM) — on a 2D robotic manipulation task. A circular agent learns to push a T-shaped block to a goal pose using receding-horizon control. Pick a model, position the block, and watch them run.

`Python` `PyTorch` `FastAPI` `React` `TypeScript` `Docker` `Pymunk`

### [Closed-Loop Arm Control](https://github.com/PMurph09/closed-loop-arm-control)
Simulation comparing MPC and reinforcement learning (DQN, PPO) for planar N-link robot arm goal-reaching. Includes a neural-network learned dynamics model for sim-to-real transfer experiments and a Three.js frontend with live parameter tuning.

`Python` `PyTorch` `stable-baselines3` `FastAPI` `React` `Three.js` `Docker` `Gymnasium`
